### 题目：一个数如果恰好等于它的因子之和，这个数就称为"完数"。例如6=1＋2＋3.编程找出1000以内的所有完数。
```java
l = []
for a in range(1,1001):
    x = a
    j = []
    sum0 = 0
    for i in range(1,x):
        if x % i == 0:
            j.append(i)
    for n in j:
        sum0 = sum0 + n
    if sum0 == a:
        l.append(a)

print(l)
```
