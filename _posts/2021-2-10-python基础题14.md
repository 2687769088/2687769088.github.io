## 题目：将一个正整数分解质因数。例如：输入90,打印出90=2*3*3*5。


```java
n = int(input())
print('{} = '.format(n), end=" ")
if not isinstance(n, int) or n <= 0:
    print('请输入一个正确的数字 !')
    exit(0)
elif n == 1:
    print('{}'.format(n))
while n not in [1]:
    for x in range(2, n + 1):
        if n % x == 0:
            n = n // x
            if n == 1:
                print(x)
            else:
                print('{} *'.format(x), end=" ")
            break


```
