## 题目：将一个正整数分解质因数。例如：输入90,打印出90=2*3*3*5。


```java
n = int(input())
print('{} = '.format(n), end=" ")
if not isinstance(n, int) or n <= 0:
    print('请输入一个正确的数字 !')
    exit(0)
elif n == 1:
    print('{}'.format(n))
while n not in [1]:
    for x in range(2, n + 1):
        if n % x == 0:
            n = n // x
            if n == 1:
                print(x)
            else:
                print('{} *'.format(x), end=" ")
            break


```
换个简单的解法
```java
a = int(input())
x = a
i = 2
j = []
count = 0
while i <= x:
    if x % i != 0:
        i = i + 1
    else:
        x = x / i
        j.append(i)
        count = count + 1
        j.append('*')
del j[-1]
if len(j) == 1:
    print("这不是素数")
else:
    print(a,"=",end='')
    for n in j:
        print(n, end='')
```
