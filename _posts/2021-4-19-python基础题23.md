### 题目：打印出如下图案（菱形）:
```java
   *
  ***
 *****
*******
 *****
  ***
   *
```

```java
for i in range(4):
    print(' ' * (7//2 - i) + '*' * (i * 2 + 1))
for j in range(3):
    print(' ' * (j + 1) + '*' * (7 - (j + 1) * 2))
```
## 推广到任意行n（n为奇数）
```java
n = int(input('输入奇数'))
for i in range(n//2+1):
    print(' ' * (n//2 - i) + '*' * (i * 2 + 1))
for j in range(n//2):
    print(' ' * (j + 1) + '*' * (n - (j + 1) * 2))
```
