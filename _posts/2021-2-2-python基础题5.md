## 题目：输入三个整数x,y,z，请把这三个数由小到大输出。
```java
a = input()
b = input()
c = input()
if a > b:
    if a > c:
        if c > b:
            print(f"{a}>{c}>{b}")
        else:
            print(f"{a}>{b}>{c}")
    else:
        print(f"{c}>{a}>{b}")
elif (c > b) and (b > a):
    print(f"{c}>{b}>{a}")
elif (b > a) and (a > c):
    print(f"{b}>{a}>{c}")
elif (b > c) and (c > a):
    print(f"{b}>{c}>{a}")




```
